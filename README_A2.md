PG29\_Vi\_WebAppFrameWorks\_A1



This is a simple Vue single page app



It has:

* Landing Page with a Counter Component
* Leaderboard

 	- Can reset leaderboard

 	- Can add new players random score

* Contact page

 	- Can only submit feedback if all input fields are filled out

* About Page



To run:

* Make sure vite is installed on front end

npm instal vite@latest

* Make sure tsx is installed on back end

npm instal tsx

* npm run dev     on back end then front end
